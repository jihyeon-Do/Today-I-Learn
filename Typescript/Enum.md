# Enum

## 1. 숫자형 열거

 열거형 (Enum)
 
 * 의미있는 상수 자료를 정의할 수 있다.
 * 키를 값에 할당 순서가 없는 집합이자 자료구조이다.
 * enum 키워드 + 파스칼케이스 조합으로 생성
 * 계산된 값을 사용할 수 있다.
    - 타입스크립트가 알아서 추론

```typescript
enum Order {
    // 숫자 열거형
    First = 1,
    Second, // 2
    Third // 3
}
```

## 2, 문자형 열거

- 숫자형과는 다르게 이넘 값 전부 다 특정 문자 또는 다른 이넘 값으로 초기화 해줘야 한다.
- 자동으로 증가되지 않는다.
